O uso das Tabelas

Obs: Não é indicado usar Tabelas para construir o Layout de um site!

Hierarquias de Tabelas

Obs:Nossas tabelas precisam obedecer uma Hierarquia:

1- Table= Tabela

2- Table Row (tr)= Linhas da Tabela

3- Table Header= Cabeçalho da Tabela

4- Table Data (td)= Dados da Tabela
